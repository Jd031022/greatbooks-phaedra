## Project Overview

This website simulates a digital archive for Racine's 1677 French classical tragedy *Phaedra*. It presents the play through multiple interactive sections with an ancient aesthetic that reflects the work's classical origins.

## Features

- **Five Interactive Sections:**
  - Home: Introduction to the play and key facts
  - Literary Analysis: Examination of structure, form, and psychological depth
  - Characters: Interactive character cards with detailed analysis
  - Themes: Exploration of philosophical and thematic elements
  - Creators: Information about website developers and project details

- **Interactive Elements:**
  - Dark/Light mode toggle
  - Ambient audio control
  - Character exploration with detailed profiles
  - Smooth navigation between sections
  - Print functionality

- **Design Features:**
  - Ancient parchment aesthetic with wax seals and ink blots
  - Responsive layout for all devices
  - Classical typography and decorative elements
  - Smooth animations and transitions
